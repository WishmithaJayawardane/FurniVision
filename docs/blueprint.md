# **App Name**: FurniVision

## Core Features:

- Room Specification Entry: Allow designers to enter and store room specifications including size, shape, and color scheme.
- 2D Design Creation: Enable designers to create new designs based on room specifications and visualize them in 2D.
- 2D to 3D Conversion: Convert 2D designs into 3D views for a realistic presentation of the furniture arrangement within the room.
- Furniture Scaling: Scale furniture items to fit room dimensions appropriately, providing a realistic spatial representation.
- Design Saving and Editing: Save completed designs for future use, with options to edit or delete existing designs as needed.
- AI-Powered Color Scheme Suggestion Tool: The app can suggest the best matching color schemes based on user selected furnitures. It reasons when the furniture should or should not be color matched

## Style Guidelines:

- Primary color: Soft blue (#A0D2EB) to evoke trust and sophistication.
- Background color: Very light blue (#F0F8FF) for a clean and spacious feel.
- Accent color: Muted orange (#E59866) to highlight key interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif, for a clean and modern user experience.
- Use consistent and minimalist icons to represent different furniture types and design options.
- Employ a grid-based layout to ensure a structured and responsive design across various screen sizes.
- Implement subtle transitions and animations for feedback, such as furniture placement confirmation and design saving.
